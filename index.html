MCCD Package Contents¶

Multi-CCD Point Spread Function Modelling.

Main contributor

Tobias Liaudat (tobias.liaudat@cea.fr)

Release date

08/10/2020

Documentation

https://cosmostat.github.io/mccd/

Repository

https://github.com/CosmoStat/mccd

This package is used to generate a PSF model based on stars observations in the field of view. Once trained, the MCCD PSF model can then recover the PSF at any position in the field of view.

Installation¶

To install using pip run the following command:

$ pip install mccd

To clone the MCCD package from GitHub run the following command:

$ git clone https://github.com/CosmoStat/mccd.git
$ cd mccd
$ python setupy.py install

Check that the installation of PySAP went correctly by launching:

$ python setupy.py test
